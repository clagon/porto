# 安心・安全に使うためのガイド & よくある質問 (FAQ)

「ポートを開けるのって、パソコンがウイルスに乗っ取られたりしないか心配...」
「設定したはずなのに、友達がどうしてもサーバーに入れないと言っている...」

そんな疑問や不安を解消し、あなたが安心して Porto（ポルト）を使いこなせるように優しく解説します。

---

## 🔒 1. セキュリティの疑問に寄り添う Q&A

### Q. ポートを開放すると、パソコンがハッキングされたりしませんか？
**A. 必要最低限のポートだけを一時的に開く仕様なので、過度に怖がる必要はありません！**
- ハッキング（不正侵入）は、泥棒が「開いている窓」から忍び込むようなものです。
- 従来のやり方（ルーター設定の固定化）は、「一度窓を開けたらずっと開きっぱなし」にするため危険でした。
- しかし、Porto が使う **UPnP** という仕組みは、**「あなたがゲームで遊ぶ時だけピンポイントで窓を開け、用が済んだら閉じる」** という一時的な開放を行います。
- アプリで「停止」を押すか、PCやルーターを再起動するだけで、開けたポート（窓）は完全に閉じられて元通りになります。

### Q. 最も安全に使うためのベストプラクティスは？
**A. 「遊ぶ時だけ開き、終わったら停止ボタンを押す」を習慣にしましょう！**
- アプリを使い終わったら、ダッシュボードのポートカード下部にある **「停止」** ボタンをクリックしてください。
- 開放ルールが即座にルーターから消去されるため、これが最も安全にネットワークを守る方法です。

## 🛠️ 2. 「友達が繋がらない！」ときの解決ストーリー

設定は成功したのに友達が接続できないときは、以下の 3 つのステップを順番に確認してみましょう。ほぼすべての原因がこのどれかに当てはまります！

### 🔍 チェック 1. PC側でゲームやアプリが「本当に起動」していますか？
- ルーターのポート（窓）を開けても、お部屋の中に **「通信を受け止める主役（ゲームサーバー等）」** がいなければ、友達は接続できません。
- 接続を試す前に、Minecraftサーバーなどのバックグラウンドアプリが正常に起動し、プレイヤーの受け入れ準備（ログの `Done` 表示など）が整っているかもう一度確認してください。

### 🛡️ チェック 2. PCの「ファイアウォール」が邪魔をしていませんか？
- **これが一番多い原因です！**
- Windows や Mac には、外部からの通信を自動的にブロックする「防御壁（ファイアウォール）」が備わっています。
- ゲームサーバーを初めて起動したときに、「このアプリの通信を許可しますか？」という画面（Windows Defender等）が出た際、チェックを入れ忘れていると通信が完全に弾かれてしまいます。
- **対策**:
  1. PCの「設定」➡️「アップデートとセキュリティ」➡️「Windows セキュリティ」から、ファイアウォールの設定を開きます。
  2. 使用しているゲーム（例: `javaw.exe` やゲームプログラム名）の通信が「プライベート」および「パブリック」で「許可」されているか確認し、許可にチェックを入れます。

### 🌐 チェック 3. ルーターの「UPnP機能」が有効になっていますか？
- アプリの画面で「ルーター未検出（ステータスがオレンジ）」になっている場合、おうちのルーターで自動ポート開放機能（UPnP）がオフに設定されている可能性があります。
- **対策**:
  1. パソコンやスマホから、お使いのルーターの設定画面（通常はブラウザで `192.168.1.1` などを入力して開きます）にログインします。
  2. 設定項目の中から **「UPnP」** または **「ユニバーサルプラグアンドプレイ」** という項目を探し、**「有効（ON）」** に切り替えて保存します。
